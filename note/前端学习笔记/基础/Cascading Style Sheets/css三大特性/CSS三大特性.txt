






		CSS三大特性：


			1.继承
				父代的字体样式会被所有子代继承
				（但a，h，div不会继承（div高不不继承且为0px，宽继承））
				

			2.层叠
				（
				处理页面冲突。优先级的表现形式。
				两个选择器为同一个元素设置了不同的属性，它们会同时作用
				两个选择器为同一个元素设置了相同的属性，它们会发生层叠（受优先级影响，如果同级或同权重，后面的覆盖前面的）
				）


			3.优先级
				单个（ !important > 行内样式 > id > class > 标签 > 通配符 > 继承 > 浏览器默认 ）  * !important 不能继承，只适用于当前标签。


			复合选择器 （优先级的权重计算：
					如果选择器混合使用，就数 id, class, 标签的数量（不看本质，也不看混合顺序，但一定要命中对应标签）。
					先比id数，再比class数，最后再比标签数。

					权重值：id > class > 标签 
				）

			权重也一样——>后面的覆盖前面的
